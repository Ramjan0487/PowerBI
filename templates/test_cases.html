Contract Management System
12 Comprehensive Test Cases · Flow Diagrams · User Cases & Actions · Problem / Solution documentation. Built with Python Flask, Power BI dashboards, SMS (Twilio) + Email (SMTP) reminders.
Flask 3.0 Chart.js 4.4 Python 3.11 Twilio SMS Flask-Mail Jinja2
👤 User Cases & Actions
Actor: Contract Manager
📄 Contract Lifecycle Manager
  • Create new contract with vendor details
  • Upload and sign contract documents
  • Monitor contract expiry dates
  • Trigger renewal or termination workflow
  • View contract value and budget utilization
Actor: Finance Officer
💳 Transaction Controller
  • Record payments and invoices against contracts
  • Approve or reject pending transactions
  • Generate financial transaction reports
  • Monitor budget usage per contract
  • Export ledger to CSV / Excel
Actor: System / Scheduler
🤖 Automated Reminder Engine
  • Scan all contracts daily for expiry
  • Send SMS to contract owner's phone
  • Send HTML email with contract details
  • Log all reminder dispatches
  • Escalate critical (≤7 day) alerts
Actor: Executive
📊 Dashboard Consumer
  • View executive KPI dashboard
  • Drill into contract status distribution
  • Review monthly portfolio value trends
  • Monitor transaction success rates
  • Export Power BI-style visual reports
Actor: Vendor
🏢 External Vendor Portal
  • Receive expiry notification by SMS/Email
  • View contract terms and status
  • Submit renewal request via portal
  • Track outstanding payment status
Actor: QA Engineer
🧪 Test & Validation
  • Execute 12 defined test cases
  • Validate SMS/Email reminder delivery
  • Test dashboard data accuracy
  • Verify API response formats
  • Document pass/fail outcomes
